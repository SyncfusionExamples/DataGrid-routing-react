# DataGrid-routing-react

## Repository Description
A small React sample demonstrating client-side routing and a DataGrid sample integration for viewing tabular data.

## Overview
This repository contains a minimal React application that demonstrates navigation between pages and a tabular data view. The app uses `react-router-dom` for routing and includes a DataGrid integration via the project dependencies to render structured data from `data.js`.

## Features
- Client-side routing between pages (Home, About, Contact, User)
- Tabular data presentation using a React DataGrid package
- Sample data provided in `data.js` for quick testing

## Prerequisites
- Node.js (12+ recommended)
- npm (or yarn)

## Install and Run
1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm start
```

Open http://localhost:3000 in your browser and navigate the app. Use the header navigation to move between pages and locate the DataGrid example to view the sample dataset.
